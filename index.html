Timeline // กำหนดการสำคัญ

  • 26-27 ก.ย. 2569 (17:00 น.): ประตูหน้าสนามหลักเปิดให้ลงทะเบียนเข้าสู่พื้นที่งานแสดง

Seating Map // ผังที่นั่งและโซนการแสดง

  • ข้อมูลโซน: กรุณาเช็กรายละเอียดหมายเลขแถวระบุที่นั่งให้เรียบร้อยที่จุดจำหน่ายบัตรออนไลน์

Ticket Pickup // ข้อตกลงการรับบัตร

  • รับบัตรหน้างาน: โปรดเตรียมเอกสารยืนยันคำสั่งซื้อพร้อมบัตรประชาชนมาแสดงล่วงหน้า 3 ชั่วโมงก่อนการแสดง

Benefits // สิทธิพิเศษตามเงื่อนไขราคา

  • Soundcheck Pass: สงวนสิทธิ์สำหรับสุ่มผู้โชคดีและผู้ถือบัตรประเภทวีไอพีเต็มจำนวนเท่านั้น

Rules // กฎระเบียบความปลอดภัย

  • อุปกรณ์บันทึกภาพ: ห้ามนำกล้องถ่ายรูปสเปกสูง เลนส์ซูมระยะไกล หรือแท็บเล็ตขนาดใหญ่เข้ามาบันทึกภาพการแสดง
  • อุปกรณ์เชียร์: อนุญาตให้ใช้เฉพาะแท่งไฟ Official Lightstick ของวง และป้ายเชียร์ขนาดกะทัดรัดไม่เกินกระดาษ A4
  • มาตรการความปลอดภัย: ห้ามนำอาหาร ของขบเคี้ยว หรือเครื่องดื่มทุกชนิดผ่านประตูตรวจสัมภาระเข้ามาด้านในฮอลล์คอนเสิร์ต
